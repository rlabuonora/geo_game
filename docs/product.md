# Product

## Prototype Goal

Build a first playable geography prototype for kids. The screen should feel bright and simple, with one obvious action: start a round.

## Current Round Loop

1. The player opens the app.
2. A random country can be highlighted on the map.
3. The player presses `Start Round` to roll a new highlighted country.
4. The country card and progress indicator update immediately.

## UI Requirements

- Large country card
- Clear `Start Round` button
- Progress indicator
- Confetti placeholder for future win feedback
- Visible map with a highlighted active country
